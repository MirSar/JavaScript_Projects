Example 1: Normal function

This example shows the syntax of a function,
without the use of arrow function syntax.

function greetings() { document.getElementById("code1").style.color="black"; }

Example 2: Arrow function

This example shows the syntax of an arrow function.

greetings=() => document.getElementById("code2").style.color="black";

This example shows the syntax of an arrow function
with more than one statement.

greetings3=() =>{ document.getElementById("code3").style.color="black"; document.getElementById("code3b").innerHTML="we need {curlies} for this"; }
